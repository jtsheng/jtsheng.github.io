Courses and Electives

Recommended Courses to Explore

While all 11 courses for the major are required, there are some options when it comes to the electives. Out of the 11 electives, only 3 are required from the list. These are some of my personal choices!

DATA 320 - Machine Learning

This course covers predictive modeling techniques and exposes students to current practices and technologies that businesses use to turn data into useful information. Topics include data mining and text and web analytics. Python will be the predominant language in this course.

DATA 210 - Data Visualisation

This course provides an introduction and hands-on experience into transforming data into visualised reports. By using certain visualisation softwares such as MySQL and Tableau, students will be able to organise and visualise quantitative and qualitative data in understandable formats.

DATA 305 - Advanced Quantitative Analysis

This course involves the study of theories and practices of diverse quantitative methods and procedures relating to finance, marketing, operations, corporate strategy, and providing business analytics. Topics include linear and logistic regression modelling, optimisation, and computer simulations.
